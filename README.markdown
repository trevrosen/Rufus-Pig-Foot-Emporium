# Rufus's(s) Pig Foot Emporium

A fun demo app for RedisCacheable

## Setup

     git clone git@github.com:trevrosen/Rufus-Pig-Foot-Emporium
     cd <CLONE DIRECTORY> && bundle install
     bundle exec ruby app.rb
